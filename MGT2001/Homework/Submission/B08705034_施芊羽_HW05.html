6.91¶

$P(BAC \text{ is greater than } 0.09) = 12 \% = P(BAC \text{ is greater than } 0.09 | \text{Crash with fatality}) + P(BAC \text{ is greater than } 0.09 | \text{Crash without fatality}) \\= 0.084 + P(BAC \text{ is greater than } 0.09 | \text{Crash without fatality}) \\ \Rightarrow P(BAC \text{ is greater than } 0.09 | \text{Crash without fatality}) = 0.12 - 0.084 = 0.036.$

The probability of a crash with at least one fatality if a driver drives while legally intoxicated()$BAC \gt 0.09$ is $$\frac{P(BAC \text{ is greater than } 0.09 | \text{Crash with fatality})}{P(BAC \text{ is greater than } 0.09 | \text{Crash with fatality}) + P(BAC \text{ is greater than } 0.09 | \text{Crash without fatality})} = \frac{0.084}{0.12} = 0.7$$
